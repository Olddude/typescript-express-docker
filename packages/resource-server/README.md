# typescript-express-docker template

## Purpose

quickstarning an typescript express api that can call other apis over http and easily be deployed in any cloud environment

### Install

```sh
yarn install
```

### Test

```sh
yarn test
```

### Lint

```sh
yarn lint
```

### Build

```sh
yarn build
```

### Create self signed certificates

```sh
yarn cert
```

### Start

```sh
yarn start
```
